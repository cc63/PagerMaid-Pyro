# rate插件增强

### 1.更新数据来源为OpenExchangeRates
需自行申请API，每月免费1000条额度
本插件每2小时更新一次汇率，绰绰有余
官网：https://openexchangerates.org
### 2.支持调用bc进行金额计算
    例如：,rate usd jpy (37.95*12+9.99)/6
### 3.默认兑CNY
    例如：,rate usd 50
### 4.添加默认金额为100
    例如：,rate usd
